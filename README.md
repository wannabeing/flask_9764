## 🚀 개발환경

- Flask, Javscript, CSS, BootStrap 등을 사용
---

## ⚙️ 기능

- **인증**

  - kakao OCR API를 이용해 이미지 업로드
  - 이미지의 주민등록번호 부분만 추출하여 유효성 검사 
  - SQLite에 DB 업데이트

- **게시판**

  - summernote 라이브러리로 EDIT 페이지 구성
  - 점프 투 플라스크[](https://wikidocs.net/book/4542)를 통해 구축
